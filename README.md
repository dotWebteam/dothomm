# DotHeroes
This is frontend part of the simple Heroes of Might and Magic-like game for educational project in university based on create-react-app with Typescript and Redux.

## Setting up locally
To set up this project locally all you need to do is enter this commands:
`npm install` and `npm start`

## Try it online
Application is deployed with GH Pages here: https://dotwebteam.github.io/dothomm/
